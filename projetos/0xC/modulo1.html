Hello World e a Estrutura Básica

Módulo 0x01: O seu primeiro contato com a linguagem C.

O que é a Linguagem C?

Criada na década de 1970, C é a mãe da maioria das linguagens modernas. Ela é rápida, leve e tem acesso direto ao hardware do computador. É por isso que sistemas operacionais (como Windows e Linux) e sistemas embarcados (como o Arduino e o ESP32) são escritos nela.

O seu primeiro programa

A tradição manda que o primeiro programa de qualquer desenvolvedor seja fazer o computador dizer "Olá, Mundo!". Em C, a estrutura mínima para fazer isso é a seguinte:

// 1. Incluímos a biblioteca padrão de entrada e saída (Standard Input Output)
#include <stdio.h>

// 2. A função principal: É aqui que o programa começa a rodar
int main() {
    
    // 3. O comando para imprimir algo na tela
    printf("Hello, World!\n");
    
    // 4. Dizemos ao sistema que tudo ocorreu bem
    return 0;
}

Entendendo cada pedaço:

  • #include <stdio.h>: O C é uma linguagem "crua". Ele não sabe ler o teclado ou escrever na tela sozinho. Essa linha importa as ferramentas que nos permitem usar o printf.
  • int main(): Todo programa em C precisa de um ponto de partida. Quando você executa o seu código, o computador procura imediatamente pela função chamada main.
  • printf: É a função que "imprime formatado" textos na tela. O \n no final serve para pular uma linha (como apertar o Enter).
  • return 0;: É a forma de o programa avisar ao sistema operacional: "Terminei o meu trabalho e não houve nenhum erro (zero erros)".
