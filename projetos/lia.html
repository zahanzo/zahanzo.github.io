Local Intelligent Assistant — Arquitetura e Implementação

Python 3.10+ FastAPI + SQLite Multi-Provider LLM Open Source

Visão Geral

L.I.A (Local Intelligent Assistant) é um assistente de IA modular e extensível que roda localmente, oferecendo controle total sobre dados e processamento. O sistema foi projetado com uma arquitetura multimodal que integra visão computacional, processamento de voz, memória semântica de longo prazo e um sistema de plugins extensível.

Diferente de assistentes cloud-based, L.I.A prioriza privacidade, customização e controle total do usuário sobre o comportamento e dados do sistema. Todo o processamento pode ocorrer localmente, com suporte opcional a provedores de IA externos quando necessário.

15+ Módulos Core

Arquitetura do Sistema

A arquitetura de L.I.A foi construída seguindo princípios de modularidade e separação de responsabilidades. Cada componente é independente e se comunica através de interfaces bem definidas.

🎤

Speech-to-Text com OpenAI Whisper e Vosk. Suporta modelos locais e processamento em tempo real.

🧠

Interface web moderna com FastAPI. Dashboard de controle, gerenciamento de plugins e histórico.

memory.py — Memória Semântica RAG

Sistema de memória de longo prazo usando sentence-transformers para gerar embeddings multilíngues. Busca semântica baseada em cosine similarity para recuperar contexto relevante. Implementa decay automático para manutenção periódica. 

# Geração de embeddings multilíngues
from sentence_transformers import SentenceTransformer

_embedding_model = SentenceTransformer(
    "paraphrase-multilingual-MiniLM-L12-v2"
)

def search_memories(query: str, top_k: int = 5):
    """Busca semântica com cosine similarity"""
    query_vec = _generate_embedding(query)
    
    # Calcula similaridade com todas as memórias
    results = []
    for memory in all_memories:
        similarity = _cosine_similarity(query_vec, memory.embedding)
        results.append((memory, similarity))
    
    return sorted(results, key=lambda x: x[1], reverse=True)[:top_k]

server_mcp.py — Sistema de Ferramentas

Implementa o Model Context Protocol (MCP) para exposição de ferramentas à IA. Sistema de "Masters" permite agrupar ferramentas relacionadas e ativá-las contextualmente. Suporta execução de código Python síncrono e assíncrono. 

# Sistema de Masters - Ferramentas Contextuais
def get_active_tools():
    """Retorna ferramentas globais + ferramentas do master ativo"""
    active_master = get_active_master()
    
    # Ferramentas GLOBAIS (sempre disponíveis)
    global_tools = run_db(
        "SELECT * FROM mcp_tools WHERE parent_id IS NULL"
    )
    
    if active_master:
        # Ferramentas CONTEXTUAIS (só quando master ativo)
        contextual_tools = run_db(
            "SELECT * FROM mcp_tools WHERE parent_id = ?",
            (active_master,)
        )
        return global_tools + contextual_tools
    
    return global_tools

webui.py — Interface Web FastAPI

Interface web moderna construída com FastAPI. Oferece dashboard completo para gerenciamento de configurações, histórico de conversas, plugins, memórias e personas de roleplay. Comunicação em tempo real via arquivos JSON. 

# API REST com FastAPI
app = FastAPI(title="L.I.A Panel")

@app.post("/api/enviar")
async def send_message(msg: Message):
    """Recebe mensagem e arquivos do frontend"""
    payload = {
        "texto": msg.texto,
        "arquivos": [
            {"url": f.url, "type": f.tipo, "name": f.nome}
            for f in msg.arquivos
        ]
    }

Funcionalidades Avançadas

Multi-Provider LLM

Suporte a múltiplos provedores de IA: OpenAI, Groq, OpenRouter e modelos locais via Ollama. Troca dinâmica entre provedores sem reinicialização.

Sistema de Plugins

Arquitetura extensível via MCP. Crie ferramentas customizadas em Python com execução síncrona ou assíncrona. Sistema de "Masters" para agrupamento contextual.

Memória RAG

Memória semântica de longo prazo com embeddings multilíngues. Extração automática de fatos das conversas e busca vetorial para contexto relevante.

Visão Computacional

Análise de tela com modelos multimodais. Monitoramento de background opcional para detecção automática de mudanças de contexto visual.

STT & TTS

Speech-to-Text com Whisper (local e cloud). Text-to-Speech com Edge TTS e ElevenLabs. Sincronização labial opcional para avatares.

Roleplay & Personas

Sistema completo de personas e cenários de roleplay. Suporte a contextos NSFW com ativação/desativação dinâmica sem limpar histórico.

Modos de Operação

Focus Mode para eliminar interrupções. Do Not Disturb com detecção automática de reuniões. Watchdog para proatividade periódica.

Internacionalização

Suporte completo a múltiplos idiomas (PT-BR, EN-US). Sistema i18n robusto para interface e respostas do assistente.

Hot Reload

Sistema de recarga de configurações em tempo real. Modificações aplicadas sem reinicialização do servidor ou perda de estado.
