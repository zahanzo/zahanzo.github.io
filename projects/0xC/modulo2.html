Variáveis e Tipos de Dados

Módulo 0x02: Como o computador guarda e lembra das coisas.

O que é uma Variável?

Imagine a memória do computador (RAM) como um gigantesco armário cheio de gavetas. Uma variável é como colar uma etiqueta com um nome em uma dessas gavetas para guardar um dado lá dentro e poder usá-lo depois.

Porém, a linguagem C é "fortemente tipada". Isso significa que você precisa dizer ao computador exatamente qual tipo de coisa você vai guardar na gaveta antes de usá-la, para que ele reserve o espaço do tamanho correto.

int (Inteiros)

Guarda números inteiros sem vírgula, positivos ou negativos. Ex: 10, -5, 1024.

float (Decimais)

Guarda números quebrados (ponto flutuante). Ex: 3.14, 99.90. Lembre-se: em C, usamos ponto em vez de vírgula!

char (Caracteres)

Guarda apenas uma única letra ou símbolo. Ex: 'A', 'x', '@'. Sempre usamos aspas simples.

Criando Variáveis na Prática

#include <stdio.h>

int main() {
    // DECLARANDO as variáveis (Criando as gavetas)
    int idade;
    float temperatura;
    char inicial_do_nome;
    
    // ATRIBUINDO valores (Colocando coisas nas gavetas)
    idade = 25;
    temperatura = 36.5;
    inicial_do_nome = 'R';
    
    // Podemos declarar e atribuir na mesma linha!
    int nivel_bateria = 100;
    
    printf("Bateria atual: %d%%\n", nivel_bateria);
    
    return 0;
}

Imprimindo Variáveis (Os Modificadores)

Para exibir o valor de uma gaveta usando o printf, nós usamos um "código secreto" (chamado especificador de formato) dentro do texto, que avisa o computador que ali entrará uma variável:

  • %d: Usado para int (Decimal Integer).
  • %f: Usado para float.
  • %c: Usado para char.
